# 🧠 Proyecto TCGConnect

Plataforma full-stack para coleccionistas TCG (Trading Card Games).  
Permite gestionar colecciones, comparar precios de tiendas y conectar con otros usuarios para intercambio o venta.

## 🔧 Estado actual

- Backend Node.js funcional (Express + JWT)
- Frontend React/Vite funcionando con fetch al API local
- Persistencia en JSON local (colecciones, usuarios)
- Codificación UTF-8 corregida
- Repositorio sincronizado en GitHub
